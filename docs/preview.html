Rescue Robot · ROS 2

Mapeo del control PS4 en joy

Documento técnico para registrar qué índice de axes[] y buttons[] corresponde a cada entrada física del control PS4.

ROS 2 joy_node sensor_msgs/msg/Joy Control PS4

Comandos de verificación

Terminal 1

ros2 run joy joy_node

Terminal 2

ros2 topic echo /joy

Notas técnicas

joy_node publica el estado del control en el tópico /joy.

El mensaje contiene dos arreglos principales: axes[] y buttons[].

Los índices usados por ROS 2 empiezan en 0.

Los valores de axes[] trabajan aproximadamente entre -1.0 y 1.0, y pueden aparecer con muchos decimales, incluso hasta 16 decimales.

Observaciones

L2

Entrada Índice Uso
L2 analógico axes[2] Mide cuánto se presiona el gatillo.
L2 digital buttons[6] Se activa cuando L2 está completamente presionado.

R2

Entrada Índice Uso
R2 analógico axes[5] Mide cuánto se presiona el gatillo.
R2 digital buttons[7] Se activa cuando R2 está completamente presionado.
